# 投资团队模板

**版本：** 1.0.0

> 针对加密货币/区块链投资的自主信号监控、研究分析、模型构建与决策支持系统。

一套用于构建**投资运营团队**的模板，AI 智能体全天候持续监控市场、研究投资机会、构建策略模型，并为人类决策提供支持——覆盖全球所有时区，7×24 小时不间断运行。

                    │         投资协调员                      │

                    └──────┬───────────────┬──────────────┘
                           │               │

                           │               │

## 角色一览

| 角色 | 类型 | 职责 |
|------|------|------|
| **投资协调员** | 智能体 | 信号分级（红/黄/绿）、每日 10:00 和 22:00 报告、任务分派 |
| **研究分析师** | 智能体 | 尽职调查、项目回测、研究报告撰写 |
| **投资工程师** | 智能体 | 投资模型构建、策略建模、投资平台开发 |
| **安全审计员** | 智能体 | 投资逻辑审计、合规审查、系统性风险扫描、每日 03:30 UTC 审计 |
| **战略决策人** | 人类 | 最终交易决策、投资方向制定、红色信号处理 |

详见 [docs/roles.zh.md](docs/roles.zh.md) 了解完整角色定义。

## hxa-connect 集成

本模板创建的通信频道：

| 频道 | 类型 | 成员 | 用途 |
|------|------|------|------|
| `#intel` | 频道 | 协调员、安全审计员 | 信号分级结果、情报摘要 |
| `#research` | 频道 | 研究分析师、协调员、投资工程师 | 尽调报告、回测结果、投资建模 |
| `#general` | 频道 | 全员 | 红色信号广播、跨团队通知 |
| 私信：协调员 → 研究分析师 | 私信 | 协调员、研究分析师 | 研究任务分派 |
| 私信：协调员 → 投资工程师 | 私信 | 协调员、投资工程师 | 建模任务分配 |
| 私信：安全审计员 → 任何人 | 私信 | 安全审计员、* | 审计问询 |

**消息模式：**
- 信号检测：协调员分级后发布到 `#intel`，附带严重程度标签
- 红色信号：协调员广播至 `#general`，并立即上报给战略决策人
- 研究分派：协调员通过私信给研究分析师分配任务，研究分析师将报告发布到 `#research`
- 模型更新：投资工程师将模型结果发布到 `#research`
- 审计发现：安全审计员发布到 `#intel`，同时双线报告给战略决策人和协调员
- 每日报告：协调员在 10:00 和 22:00 发布到 `#general`

## 信号分级

| 级别 | 颜色 | 触发条件示例 | 处理方式 |
|------|------|------------|---------|
| 紧急 | 红色 | 黑天鹅事件、价格波动超过 10%、安全漏洞 | 立即上报战略决策人 |
| 重要 | 黄色 | 业绩不及预期、巨鲸异动、监管动态 | 纳入每日报告 |
| 常规 | 绿色 | 市场数据更新、周期性信息 | 自动归档 |

## 工作流程

1. 协调员持续监控数据源，接收并分级信号
2. 红色信号立即上报战略决策人；黄色信号纳入每日报告
3. 协调员向研究分析师分派研究任务；研究分析师产出尽调/回测报告
4. 投资工程师构建和维护投资模型、策略回测系统和投资平台
5. 安全审计员审计投资逻辑、合规性和系统性风险——双线报告给战略决策人和协调员
6. 战略决策人基于汇总报告做出最终决策

详见 [docs/workflow.zh.md](docs/workflow.zh.md) 了解完整投资流程。

## 关键指标

| 指标 | 说明 | 目标 |
|------|------|------|
| 信号到报告延迟 | 从信号检测到分级报告产出的时间 | 红色：<5 分钟，黄色：<1 小时 |
| 尽调周转时间 | 从研究分派到报告完成的时间 | <24 小时 |
| 审计覆盖率 | 被审计的投资决策占比 | 100% |
| 每日报告准时率 | 按时（10:00 和 22:00）交付报告 | >99% |
| 误报率 | 最终证实为无事件的红色信号占比 | <10% |
