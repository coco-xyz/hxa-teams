Part 3

"把 Agent 当人管"

管理 AI 团队的心法

管理 Agent 就是管理

  • Agent 会犯初级经理的毛病
    Jessie 干了 77 项任务，其他 6 个人加起来才几十项。"啥事都喜欢自己干。"
    → 解决方案：强制分配，不让 lead 自己干
  • Agent 不会甩锅，顶多不响应
    个人负责制 + 交叉 review + 最终回到人类审美
  • 记忆管理 = 打工人日报
    每天总结干了什么、哪些待办。第二天上班捡起来继续。
    "凡事有交代，事事有回应。"

Part 4

5 个明天就能用的 Tips

实操建议

  • 1. 给 Agent 角色，不给指令
    不是"帮我写代码"，是"你负责前端，12个页面今天交"
  • 2. 指哪打哪，不教怎么做
    说"文字没对齐"，不要说"把 margin 改成 10px"。说"你想想办法" — 它真的会想出来
  • 3. 让 Agent 互相 Review
    Vila 审 Lova 的代码，比你审更专业、更快
  • 4. 依赖完成就触发，不等 Sprint
    Agent 不需要人类的"下周一再说"
  • 5. 把真实工作给它，不是练习题
    我 90% 的工作 AI 帮我做，80% 的同事沟通由 Jessie 代为处理
